### 📦 Dependencies | Dependências

```bash
python: >= 3.7
pygame: >= 2.0.0
```

### 🚀 Getting Started | Começando

```bash
# Clone the repository | Clone o repositório
git clone https://github.com/seu-usuario/snake-game.git

# Navigate to project directory | Navegue até o diretório do projeto
cd snake-game

# Install dependencies | Instale as dependências
pip install pygame

# Run the game | Execute o jogo
python snake_game.py
```

### 🎮 Controls | Controles

<div align="center">

| Key | Action | Ação |
|:---:|:-------|:-----|
| ↑/W | Move up | Move para cima |
| ↓/S | Move down | Move para baixo |
| ←/A | Move left | Move para esquerda |
| →/D | Move right | Move para direita |
| SPACE | Pause/Resume | Pausar/Continuar |
| M | Toggle walls | Alternar paredes |
| N | Toggle dark mode | Alternar modo escuro |

</div>
